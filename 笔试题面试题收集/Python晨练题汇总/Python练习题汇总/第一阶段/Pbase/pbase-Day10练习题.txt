练习1. 现有如下程序：
    def  myfun1(a):
        global a
        a = 111
        b = 222
        def myfun2(b):
            nonlocal b

    myfun1(3)
    运行此程序会出错吗？为什么会出错？

会出错
因为函数中的形参名字不能和 global 变量名 或者 nolocal 变量名 的名字相同
