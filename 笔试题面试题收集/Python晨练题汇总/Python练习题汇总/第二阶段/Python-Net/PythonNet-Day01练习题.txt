4. 
    (1).在多进程中，父进程默认会等子进程吗？  
       会

    (2).当子进程加了join()函数，父进程会等子进程吗？和(1)中有什么不同？
       会， 主进程遇到join()会进入阻塞状态，直到子进程结束才解除阻塞继续运行

    (3).利用os.fork()函数，父进程会等子进程吗？
       不会
